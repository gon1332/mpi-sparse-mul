void coo_matrix_vector_mul ( const T  A,
const double *  x,
double *  y 
)

Multiplies matrix A by vector x and stores the result in vector y.

Parameters
A[in]Input matrix
x[in]Input vactor
y[out]Result of

Performs a matrix-vector multiplication that fits the COO format of sparse matrices. If the coo matrix has nnz elements and the names below for the arrays that represent it:

  • values - array of non-zero values from original matrix
  • row_ind - array of row indices corresponding to the rows of the original matrix
  • col_ind - array of row indices corresponding to the columns of the original matrix

the algorithm used is:

1 for i in 1..nnz:
2  y[row_ind[i]] = y[row_ind[i]] + values[i] * x[col_ind[i]]
Precondition
First create and initialize a coo matrix with coo_matrix_new.
